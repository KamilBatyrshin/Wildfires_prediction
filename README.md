Программное обеспечение для прогнозирования площади и длительности пожара в Российской Федерации.

Данные для обучения взяты с https://tochno.st/datasets/fires

analysis_less_outliers.ipynb - юпитерский питон-скрипт с предварительной обработкой и анализом данных, обучением и анализом моделей.

Презентация и отчёт - презентация по программному проекту и отчёт о разработке.

Остальные файлы - само программное решение для прогнозирования площади и длительности пожара.
